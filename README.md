# Community Library DOM Assessment

## Instructions 

- Run `npm install` before you begin working so the test dependencies are  available.
- Do not modify anything inside the `tests/` folder.
- The file you must edit is `script.js`.
- Do not modify any lines of code if you're directed not to.
- 
- You can use `npm test` to check your work, but it is optional.

## Project files

- `index.html` — page structure and DOM elements
- `style.css` — page styling
- `script.js` — the file to complete for the assessment
- `tests/script.test.js` — automated test suite (do not change)


## Submission 
- To push : 
    - Run : git add . 
    - Run : git commit -m "Your commit message"
    - Run : git push origin main
- Push your work to GitHub and paste the HTTPS link on the allocated space in your question paper. 
- Submit your question paper on your Learner Dashboard.
